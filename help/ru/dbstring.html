dbstring
Index

DBString

Формат:

dbstring( номер_колонки )

Описание:

Возвращает строку из указанной колонки результата SQL-запроса к базе. Больше информации о базах данных и, в частности, об SQLite можно найти на домашней странице SQLite http:sqlite.org и странице SQL на Wikipedia http:ru.wikipedia.org/wiki/SQL.

Смотри также:

DBClose, DBCloseSet, DBExecute, DBFloat, DBInt, DBOpen, DBOpenSet, DBRow
