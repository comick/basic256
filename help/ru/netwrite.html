Index

NetWrite

Формат:

netwrite string
netwrite( строка )
netwrite номер_сокета, строка
netwrite( номер_сокета, строка )

Описание:

Посылает строку, заданную параметром строка в открытое сетевое соединение. Если номер_сокета, используется нулевой (0) номер.

Смотри также:

NetAddress, NetClose, NetConnect, NetData, NetListen, NetRead

Пример использования смотри на странице NetConnect.
