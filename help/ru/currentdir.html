Index

Currentdir

Формат:

currentdir
currentdir( )

Описание:

Возвращает полный путь к рабочему каталогу программы BASIC-256. Для всех систем (включая Windowstm) прямой слеш (/) используется для разделения имен каталогов в полном пути.

Смотри также:

Changedir, Close, Eof, Exists, Kill, Open, Read, Readline, Reset, Seek, Size, Write, Writeline

Впервые в версии:

0.9.6r
