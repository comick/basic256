Index

NetClose

Формат:

netclose
netclose( )
netclose номер_сокета
netclose( номер_сокета )

Описание:

Закрывает текущее сетевое соединие (сокет). Если номер_сокета не указан, используется нулевой (0) номер.

Смотри также:

NetAddress, NetConnect, NetData, NetListen, NetRead, NetWrite

Пример использования смотри на странице NetConnect.
