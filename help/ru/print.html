print
Index

Print

Формат:

print выражение [ ; ]

Описание:

Печатает текст в окне текстового вывода, добавляя перевод строки. Если в конце указан знак точка с запятой, символ новой строки не добавляется.

Смотри также:

Decimal

Пример: 

print "2 х 2 = ";
print 2*2
print "2 + 2 = "
print 2+2
Будет напечатано 
2 х 2 = 4
2 + 2 =
4
