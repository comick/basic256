size
Index

Size

Формат:

size
size()
size( номер_файла )

Описание:

Возвращает размер в байтах открытого файла. Если номер_файла не указан, используется номер нуль (0).

Смотри также:

Changedir, Close, Currentdir, Eof, Exists, Kill, Open, Read, Readline, Reset, Seek, Write, Writeline
