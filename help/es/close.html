Formato

close
close ( )
close NúmerodeArchivo
close ( NúmerodeArchivo )

Descripción

Cierra un archivo abierto. Si no hay ninguno abierto con ese número de archivo, este comando no hace nada. Si el número de archivo no se especifica , se usa el número cero (0).
