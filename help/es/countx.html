countx
Index

Countx

Formato

countx ( pajar , ExpReg )

Descripción

Devuelve el número de veces que la expresión regular ExpReg aparece en la cadena pajar.

Ejemplo 

print countx("Hello", "[hH]")
print countx("Buffalo buffalo buffalo.","[Bb]uffalo")
muestra: 
1
3
