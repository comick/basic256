While / End While

Formato

while ExpresiónBooleana
instrucción(es)
end while

Descripción

Ejecuta la(s) instrucción(es) dentro del bucle WHILE hasta que ExpresiónBooleana se evalúe como falso. El bucle WHILE / END WHILE ejecuta las instrucciones cero o más veces. La evaluación se hace antes de que el código que está dentro del bucle sea ejecutado.

Ver también

Do / Until, For / Next

Ejemplo 

r = 1
while r < 6
  print r
  r = r + 1
end while
mostrará: 
1
2
3
4
5
