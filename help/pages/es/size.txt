===== Size =====
==== Formato ====
**size**\\
**size()**\\
**size**(//NúmerodeArchivo//)
==== Descripción ====
Devuelve el tamaño en bytes de un archivo abierto. Si el número de archivo no se especifica, se usa el número cero (0).
