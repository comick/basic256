===== Close =====

==== Format ====

**close**\\
**close** ( )\\
**close** //filenumber//\\
**close** ( //filenumber// )

==== Description ====

Closes an open file.  If no file is open to that file number, this command does nothing.  If the file number is not specified file number zero (0) will be used.
