===== Sqr or Sqrt =====

==== Format ====

**sqr** ( //expression// )\\
**sqrt** ( //expression// )

==== Description ====

Returns the square root of the number //expression//.

==== Example ====

<code>
print sqr(2)
print sqrt(10)
</code>
will print
<code>
1.414214
3.162278
</code>
