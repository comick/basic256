===== OutputVisible =====

==== Format ====
**outputvisible** ( //boolean// )\\
**outputvisible** //boolean//

==== Description ====
Sets the view option for the "Text Output" window.  Allows the program to control the checkable option on the View menu of the BASIC-256 IDE.
