===== Right =====
==== Format ====
**right**( //string//, //length//)
==== Description ====
Returns a portion of the specified //string//, starting from the last //length// characters from the right end of the of the string.
==== See Also ====
[[mid|Mid]], [[left|Left]]
==== Example ====
<code>
print right("Hello", 2)
</code>
will display
<code>
lo
</code>
