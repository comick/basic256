Command Color
Format color colorname
color red, blue, green
Description Sets the current drawing color to colorname or to RGB color red, blue, green.

Example 
clg
color red
circle 100,100,50
color 255,160,160
circle 100,100,25
Will draw a red circle with a pink center.
