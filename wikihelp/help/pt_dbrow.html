pt:dbrow

DBRow

Formato

dbrow
dbrow ( )
dbrow ( DatabaseNumber )
dbrow ( DatabaseNumber , DataSetNumber )

Descrição

Avança para o próximo registo. Devolve true se existe um registo e false se se não existem mais registos.
