fr:right

Right

Format

right( string, length)

Description

Retourne length caractères la partie droite de la chaîne de caractères string.

Voir Aussi

Mid, Left

Exemple

print right("Bonjour", 4)

Affichera 

jour

Disponible depuis la version

0.9.5b
