pt:spritedim

Spritedim

Formato

spritedim n
spritedim ( n )

Descrição

Cria n espaços de memória para sprites. As sprites são acedidas por números de 0 a n-1.

Exemplo

# cria uma sprite com o número 0
clg
fastgraphics
spritedim 1
a$="Basic 256"
Text 0,0,a$
spriteslice 0,0,0,textwidth (a$),20
spriteshow 0

# animação roda e amplia
for n=0 to 2*pi step .002
clg
spriteplace 0,150,150,n,n
refresh
next n
