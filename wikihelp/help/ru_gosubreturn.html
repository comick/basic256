ru:gosubreturn

Gosub / Return

Формат:

gosub метка
метка:

 оператор(ы)\\

return

Описание:

Выполнение программы переходит на точку определенную параметром метка. Пока не встретится команда RETURN, программа выполняет все строчки, следующие за GOSUB. Команды GOSUB могут быть вложены друг в друга.
