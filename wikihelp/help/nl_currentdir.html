Currentdir

Formaat

currentdir
currentdir ( )

Description

Geeft het volledig gekwalificeerd pad naar de werking directory van BASIC-256

Om de folders van elkaar te scheiden worden altijd Forward Slashes (/) gebruikt.

Zie ook

Changedir, Close, Eof, Open, Read, Readline, Reset, Write, Writeline, Exists, Seek, Size

Voorbeeld

print currentdir ( )

geeft dan 

/home/guest
