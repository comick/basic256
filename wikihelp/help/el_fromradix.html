FromRadix

Σύνταξη

fromradix ( συμβολοσειρά, βάση )

Περιγραφή

Επιστρέφει τον ακέραιο αριθμό που η δοσμένη συμβολοσειρά αναπαριστά στην καθορισμένη βάση.

Παράδειγμα

print fromradix("ffef",16)
print fromradix("1001101", 2)
print fromradix("a1z9",36)

θα απεικονίσει

65519
77
469125

Εισήχθηκε με την έκδοση

0.9.9.45
