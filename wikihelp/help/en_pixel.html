Pixel (Function)

Format

pixel (x_expr, y_expr )

returns rgb_expr

Description

Returns the RGB value of the pixel at the x_expr and y_expr coordinate. If the pixels has not been set since the last Clg command or was drawn with the color CLEAR a -1 will be returned.
