pt:color

Color

Formato

color colorname
color ( colorname )
color red, blue, green
color ( red, blue, green )
color rgb_value
color ( rgb_value )

Descrição

Estabelece como cor de desenho colorname; para RGB composto por red, blue, green; ou para valor RGB que pode ser obtido através dos componentes através da formula RGB = RED*(256^2) + GREEN*256 + BLUE

Existe uma cor especial CLEAR com valor -1 (transparência). Isto é particularmente importante para as sprites.

Exemplo

clg
color rgb(128,128,128)
rect 0,0,graphwidth, graphheight
color red
circle 100,100,50
color 255,160,160
circle 100,100,25

Will draw a grey rectangle with a red circle with a pink middle.
