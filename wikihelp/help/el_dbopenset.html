Περιγραφή

Εκτελεί μια εντολή SQL και δημιουργεί μια εγγραφή η οποία μπορεί να προσπελαστεί από το πρόγραμμα και να χρησιμοποιηθούν τα απότελέσματα.

Παράδειγμα

Δες πως μπορείς να χρησιμοποιήσεις την εντολή στην σελίδα DBOpen.

Δες επίσης

DBClose, DBCloseSet, DBExecute, DBFloat, DBInt, DBNull, DBOpen, DBOpenSet, DBRow, DBString. Freedb, Freedbset

Εξωτερικοί σύνδεσμοι

Περισσότερες πληροφορίες σχετικά με τις βάσεις δεδομένων και την SQLite μπορείς να βρεις στις σελίδες SQLite Home Page και SQL στην Βικιπαίδεια.
