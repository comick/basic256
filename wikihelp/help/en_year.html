en:year

Year (Function)

Format

year
year ( )

returns positive_integer_valuea non-negative whole number

Description

Returns the current system clock's 4 digit year.

Example

print "today's date is ";
print (month + 1) + "/" + day + "/" + year

will display 

today's date is 11/30/2009
