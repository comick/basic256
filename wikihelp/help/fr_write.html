Write

Format

write chainedecaracteres
write ( chainedecaracteres )
write numerofichier, chainedecaracteres
write ( numerofichier, chainedecaracteres )

Description

Écrit chainedecaracteres à la fin d'un fichier ouvert. Si numerofichier n'est pas spécifié, zéro (0) est utilisé par défaut.
