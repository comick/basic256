DBExecute

Format

dbexecute SqlStatement
dbexecute ( SqlStatement )

Description

Exécute une commande SQL sur le fichier de base de données SQLite ouvert. Cette commande ne produit pas d'enregistrement.

Exemple

Voir l'exemple de la page de DBOpen.
