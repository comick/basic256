pt:changedir

Changedir

Formato

changedir expressão
changedir ( expressão )

Descrição

Muda a directoria de trabalho para o caminho de acesso especificado na expressão. Em qualquer sistema incluindo no Windows uma barra à direita (/) será usada para separar pastas de um caminho.

Ver também

Close, Currentdir, Eof, Open, Read, Readline, Reset, Write, Writeline, Exists, Seek, Size

Introduzido na versão

0.9.6r
