en:netclose

NetClose (Statement)

Format

netclose
netclose ( )
netclose socket_numberinteger expression representing one of the current network sockets
netclose ( socket_numberinteger expression representing one of the current network sockets )

Description

Close the specified network connection (socket). If socket_numberinteger expression representing one of the current network sockets is not specified socket number zero (0) will be used.

Example

See example of usage on NetConnect page.
