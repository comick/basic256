en:input

Input (Statement)

Format

input prompt_exprstring expression used to ask the user for input, string_variable$variable used to store strings
input prompt_exprstring expression used to ask the user for input, numeric_variablevariable used to store numbers
input string_variable$variable used to store strings
input numeric_variablevariable used to store numbers
input prompt_exprstring expression used to ask the user for input, string_array_variable$variable to store an array of string values[index]
input prompt_exprstring expression used to ask the user for input, numeric_array_variablevariable to store an array of numeric values[index]
input string_array_variable$variable to store an array of string values[index]
input numeric_array_variablevariable to store an array of numeric values[index]
input prompt_exprstring expression used to ask the user for input, string_array_variable$variable to store an array of string values[index, index]
input prompt_exprstring expression used to ask the user for input, numeric_array_variablevariable to store an array of numeric values[index, index]
input string_array_variable$variable to store an array of string values[index, index]
input numeric_array_variablevariable to store an array of numeric values[index, index]

Description

Waits for the user to type a line of text into the text output window. When the user hits the enter or return key, the line is read in to a string_variable or a numeric_variablevariable used to store numbers.
User may optionally be prompted for the input by prompt_exprstring expression used to ask the user for input.
If a numeric variable is specified and non-numeric data is entered a zero will be assigned to the numeric variable.
References to array elements may also be specified.
