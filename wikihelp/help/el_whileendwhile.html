el:whileendwhile

While / End While

Format

while booleanexpression
statement(s)statement or multiple lines of statements
end while

Description

Execute the statement(s)statement or multiple lines of statements inside the while loop until the booleanexpression evaluates to false. While / End While executes the statements zero or more times. The test is done before the code in the loop is executed.

See Also

Do / Until, For / Next

Example

r = 1
while r < 6
  print r
  r = r + 1
end while

will display 

1
2
3
4
5
