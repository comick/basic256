el:netdata

NetData

Format

netdata
netdata ( )
netdata socket_numberinteger expression representing one of the current network sockets
netdata ( socket_numberinteger expression representing one of the current network sockets )

Description

Returns a true value (1) of there is data waiting to be read in using the NetRead function, else returns false (0). If socket_numberinteger expression representing one of the current network sockets is not specified socket number zero (0) will be used.

Example

See example of usage on NetConnect page.
