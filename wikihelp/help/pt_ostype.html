Ostype

Formato

ostype ( )

Descrição

Devolve um número associado ao sistema operativo.

Valores devolvidos
ValorTipo
0Windows
1Linux
2Macintosh

Exemplo

print "You are using a ";
if ostype() = 0 then
   print "windows";
else
   print "unix/linux";
end if
print " machine."

Produzirá 

You are using a unix/linux machine.

Introduzido na versão

0.9.6.58
