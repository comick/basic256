en:editvisible

EditVisible

Format

editvisible ( boolean )
editvisible boolean

Description

Sets the view option for the “Edit” portion of the main window, this is where the program is displayed and edited.

Example

editvisible 0
pause 2
editvisible 1

See Also

GraphVisible, OutputVisible

New To Version

0.9.6.65
