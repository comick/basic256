fr:getsetting

GetSetting

Format

getsetting ( program_namestring expression with a short name for this program, key_namestring expression containing a unique key representing this setting )

Description

Récupère et retourne un paramètre dans la base de registres (ou autre stockage persistant). Les chaînes de caractères program_namestring expression with a short name for this program et key_namestring expression containing a unique key representing this setting spécifient des paramètres qui doivent être les mêmes que ceux utilisés avec SetSetting (pour initialiser les valeurs). Une chaîne de caractères vide est renvoyée si le paramètre demandé n'a jamais été initialisé.
Les paramètres sauvegardés sous cette forme sont accessibles depuis d'autres programmes BASIC-256.
Cet instruction peur être désactivée à cause des risques encourues par le système si elle est mal utilisée. Sa disponibilité est configurable dans l'environnement de programmation dans le menu Édition > Préférence.

Voir Aussi

SetSetting

Exemple

setsetting "thisprogram", "testsetting", "value of setting"
print getsetting("thisprogram", "testsetting")

Affichera : 

value of setting
