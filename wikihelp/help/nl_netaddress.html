nl:netaddress

NetAddress

Formaat

netaddress
netaddress ( )

Beschrijving

De Functie geeft het IPv4 address terug van deze comuter op het locale netwerk.

Voorbeeld

print "Mijn netwerk adress is:" + netaddress
