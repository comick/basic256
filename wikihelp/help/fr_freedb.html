FreeDB

Format

freedb
freedb ( )

Description

Il est possible avec BASIC256 d'ouvrir simultanément plusieurs bases de données. La fonction freedb retourne un numéro de base de donnée disponible pour l'utilisation de DBOpen. Lorsqu'une base de données est fermée, freedb remet ce numéro dans la liste de ceux qu'il peut donner pour une prochaine ouverture.
