Writebyte

Format

writebyte byte
writebyte ( byte )
writebyte filenumber, byte
writebyte ( filenumber, byte )

Description

Écrit un octet (nombre sur 8 bit) à la fin d'un fichier ouvert. Si filenumber n'est pas spécifié, zéro (0) est utilisé par défaut.
Le fichier en question doit avoir été ouvert grâce à la fonction Openb, de cette façon le codes ASCII CR/LF ne seront pas interprétés et traduits.
