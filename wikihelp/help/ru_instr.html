ru:instr

Instr

Формат:

instr( строка, подстрока )
instr( строка, подстрока, старт_символ )
instr( строка, подстрока, старт_символ, регистр_не_важен )

Описание:

Проверяет, содержится ли подстрока в строке. Если да, то функция возвращает номер позиции первого вхождения подстроки. Если нет, возвращается нуль (0). Вы можете дополнительно указать число старт_символ, с которого нужно начинать проверку строки, а также булево значение регистра_не_важен = true, если не хотите учитывать регистр символов. Нумерация символов в строке начинается с 1.

Пример:

print instr("Привет", "вет")
print instr("101,222,333",",",5)

Будет напечатано 

3
8

Впервые в версии:

0.9.6.55
