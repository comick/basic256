pt:getsetting

GetSetting

Formato

getsetting ( program_name )

Descrição

Obtém a configuração do registry (ou outro dispositivo de armazenamento). O program_name são necessários para aceder a uma configuração pré definida . Um valor vazio ”” será devolvido se a configuração requerida não tiver sido préviamente definida.

A configuração deverá poder ser acedida por outros programas basic 256 e deverá manter-se por tempo indeterminado.

Esta instrução pode ser desligada por razões de segurança. Para tal vá a Edit>Preferences .

Ver Também

SetSetting

Exemplo

setsetting "thisprogram", "testsetting", "value of setting"
print getsetting("thisprogram", "testsetting")

will print 

value of setting
