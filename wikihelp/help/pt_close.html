Close

Formato

close
close ( )
close número de ficheiro
close ( número de ficheiro )

Descrição

Fecha um ficheiro. Se nenhum ficheiro com o número indicado está aberto então não faz nada. Ne não for especificado nenhum número de ficheiro então o valor 0 é assumido.
