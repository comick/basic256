FromHex (Function)

Format

fromhex ( string_exprstring expression )

returns integer_valuea whole number with no decimal part

Description

This function returns an integer number represented by the Hexadecimal (base 16) string. Hexadecimal represents 16 different values per digit and the symbols 0-9 and a-f are used.

Example

print fromhex("10")
print fromhex("ff")

displays

16
255
