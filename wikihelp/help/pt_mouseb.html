pt:mouseb

Mouseb

Formato

mouseb
mouseb()

Descrição

Devolve o botão que está a ser premido no rato (quando sobre a janela gráfica).Devolve zero se nenhum for premido e se vários estiverem a ser premidos devolve o total.

Return Values
ValueMouse Button Pressed
0Nenhum
1Esquerdo
2Direito
4Centro
