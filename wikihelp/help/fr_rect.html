fr:rect

Rect

Format

rect x,y,width,height
rect ( x, y, width, height )

Description

Draws a width x height pixel rectangle using the current color. The top left corner is located at x,y.

Example

color white
rect 0,0,300,300
color red
rect 50,50,150,150
color blue
rect 100,100,150,150
color green
rect 10,140,280,20
