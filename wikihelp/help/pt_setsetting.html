pt:setsetting

SetSetting

Formato

setsetting program_name, key_name, setting_value
setsetting ( program_name, key_name, setting_value )

Descrição

Guarda a configuração para um programa no registry (ou outro dispositivo de armazenamento). O program_name and key_name são necessários para guardar a uma configuração .

A configuração deverá poder ser acedida por outros programas basic 256 e deverá manter-se por tempo indeterminado.

Esta instrução pode ser desligada por razões de segurança. Para tal vá a Edit>Preferences .

Ver também

GetSetting

Exemplo

setsetting "thisprogram", "testsetting", "value of setting"
print getsetting("thisprogram", "testsetting")

will print 

value of setting
