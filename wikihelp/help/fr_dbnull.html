fr:dbnull

DBNull

Format

dbnull ( ColumnNumber )
dbnull ( DatabaseNumber , ColumnNumber )
dbnull ( DatabaseNumber , DataSetNumber , ColumnNumber )
dbnull ( ColumnName )
dbnull ( DatabaseNumber , ColumnName )
dbnull ( DatabaseNumber , DataSetNumber , ColumnName )

Description

Retourne vrai si le numéro ou le nom de colonne de l’enregistrement courant de la série d'enregistrements ouverte est une valeur NULL. Retourne faux si le champs contient une valeur
