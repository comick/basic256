en:penwidth

PenWidth (Statement)

Format

penwidth width_exprinteger expression representing width
penwidth( width_exprinteger expression representing width )

Description

Set the width of the current drawing pen. The default width is 1 pixel wide. The pen width is used in the drawing statements to define how wide the line, point, or border of a shape should be.
