ThrowError

Σύνταξη

throwerror ( ακέραιος )
throwerror ακέραιος

Περιγραφή

Προκαλεί ένα σφάλμα εκτέλεσης. Τα σφάλματα αυτά μπορούν να ανιχνευθούν με την εντολή Onerror.

Παράδειγμα

onerror errortrap
print "prin to sfalma"
throwerror 99
print "meta to sfalma"
end
subroutine errortrap()
print "Synevh to sfalma " + lasterror
end subroutine

θα απεικονίσει

prin to sfalma
Synevh to sfalma 99
meta to sfalma
