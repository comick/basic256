pt:print

Print

Formato

print expressão [ ; ]

Descrição

Escreve texto para a janela de saída de texto, acrescentando uma nova linha. Se o ponto e vírgula opcional for incluído, a nova linha não é acrescentada.

Exemplo

for n = 1 to 5
print n
next n

for n = 1 to 5
print n;
next n

mostra 

1
2
3
4
5
12345
