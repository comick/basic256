Spriteslice (Statement)

Format

spriteslice sprite_number, x_expr, y_expr, width_expr, height_expr
spriteslice ( sprite_number, x_expr, y_expr, width_expr, height_expr )

Description

Copy the rectangular region of the screen with it's top left corner represented by x_expr and y_expr of the specified height_expr and width_expr and create a sprite. The sprite will be active and movable but will not be visible until the Spriteshow statement is executed. It is recommended that you execute the Clg command before drawing and slicing the sprite. All unpainted pixels will be transparent when the sprite is drawn on the screen. Transparent pixels may also be set by drawing with the color CLEAR.
